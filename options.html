Mode:

RegEx-URL-Matchers

  • unchecked: (MANUAL-TEMP) only tabs with URLs that match with any entry on the list, will automatically be put into a temp container
  • checked: (AUTO-TEMP) all tabs, except the ones with URLs that match with any entry will open in temp containers
  • Example 1: ^https:\/\/www\.twich\.tv.*, matches all twitch.tv sites
  • Example 2: ^https:\/\/www\.youtube\.com.*, matches all youtube.com sites
